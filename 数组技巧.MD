# 数组技巧

### 交换数组中两项位置

- 使用第三个变量

```javascript
temp = arr[i];
arr[i] = arr[j];
arr[j] = temp;
```

- 异或运算 (数组中每一项都必须是 **数字**)

```javascript
arr[i] = arr[i] ^ arr[j];
arr[j] = arr[i] ^ arr[j];
arr[i] = arr[i] ^ arr[j];

// 证明
arr[i] = arr[i] ^ arr[j];

arr[j] = arr[i] ^ arr[j] ^ arr[j];
arr[j] = arr[i] ^ 0;

arr[i] = arr[i] ^ arr[j] ^ arr[i] ^ arr[j] ^ arr[j];
arr[i] = arr[j] ^ 0 ^ 0;
```

- 数组解构 (非常优雅)

```javascript
[arr[i], arr[j]] = [arr[j], arr[i]];
```
